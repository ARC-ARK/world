Atmosphere

沉浸感不是背景，而是所有關係的起點。

從版面、語氣到世界規則都圍繞同一種夜色壓力感，角色越深入，就越難保持旁觀者立場。

Roleplay

對戲不是零散片段，而是長線推進。

每次互動都可能留下標記、承諾、背叛或陣營傾向，並在之後的劇情裡回頭影響角色。

Progression

世界會前進，角色也必須表態。

權柄、覺醒、封鎖區與組織線同時運作，玩家能沿著自己的選擇長出完全不同的故事路徑。

Community

社群是場景的一部分，而不是場景之外。

設定交流、任務合作、陣營對立與情感推進會在同一個伺服器裡慢慢疊起來。

Main World / Overview

這不是單一場景，而是一套會吞沒角色立場的三界結構。

表世界維持秩序與教育信仰，交界處則堆積流亡者、覺醒者與被放逐的族群。裡世界讓慾望與血統意識變成顯性規則，而天使始終在更高的位置介入，改寫所有人對正當性的理解。

每一名角色不只在一個地圖上行動，而是在不同層級的秩序裡被迫選邊、求生、反抗或沉淪。這份世界觀的目的不是讓你閱讀完畢，而是讓你知道角色從一開始就站在壓力場裡。

Regulations / Codex

天使與神之權柄

天使並不屬於單一地理區域，而更像由人族集體信仰與秩序需求凝結出的高位存在。第四級以上的干涉，通常會被視為權柄本身的顯現。

能力分級

從微效級到戰略級，能力不只是戰力高低，而是角色被世界怎麼看待、允許介入到哪個層級的尺度。

交界處的現實

流亡、走私、異族聚居與非法覺醒都在這裡發生。它不是過渡地，而是最能看見制度縫隙的地方。

封鎖副本邏輯

副本不是獨立遊樂場，而是會把角色行為回流到世界線的壓力節點。你帶出的情報、標記與關係都會留下。

表世界

Raid / Event

封鎖副本：神眷島

Danger Level / IV

海圖之外，只有被祭儀選中的人才能踏上岸線。

神眷島是一座不存在於海圖上的封閉孤島。島內以王權、祭儀與階層構成三角結構，御柱與闢限漩渦會不定期把表世界、裡世界與交界處的來客拖進岸線。對外來者而言，這裡不是觀光地，而是一座會慢慢改寫立場、身份與離島意志的封鎖裝置。

封閉孤島 王權—祭儀—階層 藥草與香氛控制 離島線索稀缺

Participation Flow

  1. 外來者會在漩渦與岸線附近完成第一輪分流，角色需迅速決定偽裝、求援或直接潛伏。
  2. 島上存在王族、將、島民、祭儀系統與森林駐地，不同入口決定你先被誰看見。
  3. 想離島必須追查八領、禁足地與御柱等關鍵詞；想留下，則必須學會與島上權力共存。
  4. 事件結束後，副本帶出的關係、標記與立場會回流到主世界線，成為下一段劇情的壓力。

Operation Board

核心理念

反抗王權、尋找離島方式、拯救同伴或被迫同化。

推薦角色

潛伏者、談判型角色、陣營領袖、失序覺醒者。

主要場域

海岸線、森林庵、祭祀禁足地、王族居處、火山與嫡系派駐地。

地圖邏輯

部落式島嶼、無現代電子產品、藥草配方與儀式知識掌握在少數人手中。

Story Hook

劇情鉤子

王權與嫡系派並非鐵板一塊。你可以臥底、策反、營救、合作，或讓角色逐步變成局中人。

Reward

事件獎勵

副本可帶出陣營關係、角色稱號、地圖情報與關鍵道具，也能直接影響之後的組織線與世界線。

Risk

角色風險

長期停留可能導致立場鬆動、意志被削弱，甚至被結界與儀式系統慢慢同化成島嶼的一部分。
